🚀 Getting Started

This pipeline helps you generate narrative prose using Claude AI. Here's how it works:

  1. Create Characters - Define your characters with detailed profiles
  2. Build World Context - Establish your story's setting and rules
  3. Write Scene Outlines - Describe what happens in each scene
  4. Generate Prose - Let Claude create the narrative
  5. Critique & Revise - Iteratively improve with manual approval
